What is JavaScript?

  • From Wikipedia:

    JavaScript is a scripting language most often used for client-side web development. JavaScript was influenced by many languages and was designed to have a similar look to Java, but be easier for non-programmers to work with.

  • object-oriented scripting language.
  • used to work with the objects associated with a web page document — the window, the document, the elements such as forms, images, links, etc.
  • originally developed by Netscape and called LiveScript
  • netscape collaborated with Sun Microsystems on modifications to the language and it was renamed JavaScript
  • JavaScript is NOT Java
  • common uses:
    • validate form data
    • animated navigation (pop ups, drop downs, etc)
    • image gallery
    • read and write cookies
    • query client system
    • decisions, calculations

Javscript Basics

Lets review the javascript essentials:

  • embedded javascript
  • external javascript
  • javascript function calls

Implementation:

  • place Javascript code as part of an event attached to an XHTML element (inline)
  • place Javascript code between <script> tags (embedded)
  • place Javascript code in an external file (external)
  • embedded script:

    <script type="text/javascript">
    alert("Welcome to Our Site");
    </script>

  • external script:

    <script type="text/javascript" src="myjavascript.js" ></script>

    A <script> tag does not allow using both Embedded and External javascript at the same time.

  • In HTML5, only the <script> tag is required, but to maximize backward compatability, the type="text/javascript" attribute may be included
  • Oftentimes, the best way to deploy your javascript is in external file(s). Using embedded javascript reduces reusability and maintainability.
  • Note: The script tag does not permit both external and embedded within the same tag. The convention is to make all your javascript external.

Function Calls:

  • Function definition: Functions all have a name nameOfFunction, a parameter list ( ) , and one or more instructions contained within curly braces { }
    function nameOfFunction(){
	/* body of function */			
}
function nameOfFunctionWithParameters(value01, value02){
	/* body of function */			
}
  • Function Invocation: To run the code inside a function, use it's name and include any required parameters
    nameOfFunction();
nameOfFunctionWithParameters("Shania", 123);

Events:

  • more than one event can be applied to a particular tag

jQuery Basics

You will notice this same pattern in many jQuery statements, 

 $(object).event(function(){
   //javascript code
 });
  • object: any valid selector
  • event: See a list of jQuery Events
  • function: this is creates a new 'nameless' anonymous function that will be invoked as soon as the event occurs
  • javscript code: includes one or more javascript instructions, or jQuery instructions

In javascript you would assign events to trigger functions by using the event attribute, eg:
<div onclick="alert('Hello World');" ></div>
in jQuery, events are assigned in the code using the pattern noted here.

Try creating a click event for div tags

$(document).ready(function(){
	$("div").click(function(event){
		alert("Hello World");
	});
 });

Event Passing

  • If you need to know more about an event that has just occurred, define an event parameter for the anonymous function
  • Inside the function you can access the event as an object that you can query details upon
  • /* include a parameter in the anonymous function... */
$("div").click(function( event ){
	/* ...access event specific info using the parameter */
	alert("X position is " + event.pageX );
});

Overriding Default Behaviors

Sometimes you will want javascript behaviors to override the standard. For example, if you want to use the click event of an anchor tag, you may want to disable the usual anchor tag click.

Add the prevent.preventDefault();

$("a").click(function(event){
	event.preventDefault();
	alert("Hello World");
});

  • fade out

The basic pattern for applying effects is

Your instructor will walk you through some effect examples

Animate is especially powerful, and has a slightly more complex syntax:

$('selector').animate({
	property01: 'value',
	property02: 'value'
}, speed);

Note: animate can only effect CSS properties that have numeric values.

The Current Element (this)

  • You can use a dynamic selector called this to choose the element that most recently triggered an event
  • this refers to the current element
  • /* if a "div" is clicked... */
$("div").click(function(){
	/* ...you can use $(this) to choose the 
	specific div that was just clicked  */
	$(this).css('background-color', 'blue');
});
